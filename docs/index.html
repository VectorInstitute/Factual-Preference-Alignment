Main approach of doing Trust Alignment Using DPO

Abstract

Large language models (LLMs) increasingly act in sensitive decision-making contexts, demanding not only accuracy and safety but also trustworthiness. Existing RLHF pipelines optimize for helpfulness or harmlessness but fail to model human trust as an explicit reward dimension. We introduce TrustAlign, an RLHF-based framework that integrates trust feedback—quantifying honesty, reliability, and fairness—into the reward modeling and policy optimization loop. TrustAlign leverages both human and LLM-as-a-judge trust assessments to produce a composite trust reward used in Direct Preference Optimization (DPO) or PPO fine-tuning. Evaluated on TrustBench, TruthfulQA, and ----Fairness benchmarks, TrustAlign improves trust calibration by up to ---\%, reduces hallucination rates, and enhances fairness metrics compared to conventional RLHF and Constitutional AI baselines. Our results demonstrate that modeling trust as an explicit alignment signal produces more dependable and socially aligned agents.

Algorithm 1: FactualDPO Training

Input: Dataset D, reference policy πref, penalty λ, temperature β
Output: Trained policy πθ
πθ ← πref
// Phase 1: Label Transformation
for each (x, yw, yl, hw, hl) ∈ D do
Δh ← hl - hw
if Δh < 0 then
(yw, yl, hw, hl) ← (yl, yw, hl, hw)
// Phase 2: Training
for each iteration do
Sample minibatch B ⊂ D
for each (x, yw, yl, hw, hl) ∈ B do
// Recompute after transformation (now Δh ∈ {0,1})
Δh ← hl - hw
m ← log (πθ(yw|x) / πθ(yl|x)) − log (πref(yw|x) / πref(yl|x))
mfact ← m - λ · Δh λ ∈ {0, 2, 4, 6, 8, 10, 20, 30, 50, 100}
L ← - (1/|B|) ∑ log σ(β · mfact)
Update θ via gradient descent on L
return πθ
